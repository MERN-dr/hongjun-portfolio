# 🙋🏻‍♂️ 포트폴리오

## 프로젝트 내용
개인 포트폴리오를 Next Js 와 Typescript 그리고 TailwindCSS를 사용하여 제작하였습니다.


## 주요 기능
- 낮과 밤
- cursor 애니메이션
- scroll 애니메이션
- 이메일
- 한/영 변경
